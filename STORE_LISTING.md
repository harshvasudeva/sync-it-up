## Full Description (for Chrome Web Store)

### Headline
**Stop juggling tabs across browsers. See every open tab, everywhere, in one place.**

### Overview
SyncTabs brings order to browser chaos. Install it in Chrome, Edge, Brave, Vivaldi, Opera—or all of them—and instantly see every tab you have open across all browsers. Click any tab to open it in your current browser. Everything syncs in real time.

**Zero setup. Zero cloud. Zero account. Everything stays on your machine.**

### What It Does

#### View All Your Tabs
- See tabs from **all your browsers** in a single popup
- Tabs grouped by window and browser
- Live tab counts and active tab indicators

#### Sync Across Browsers
- Open a tab in one browser, access it from another
- Click any remote tab to open it instantly
- See what's open even when browsers are closed

#### Tabs That Persist
- Tabs saved locally even after browser close
- Your tab history available anytime

### How It Works

**Step 1: Install the Extension**
Add SyncTabs to each browser you use.

**Step 2 (Optional): Install Companion App**
For cross-browser syncing, download the lightweight Companion app (Windows, macOS, Linux). It runs locally on your machine—no cloud connection.

**Step 3: Start Browsing**
Click the SyncTabs icon to see all your tabs. Click any tab to open it instantly.

### Permissions Explanation

**Why we request "tabs" permission:**
- To read which tabs you have open and their titles/URLs
- Used only for displaying tabs in the popup
- Not shared with servers or third parties

**Why we request "storage" permission:**
- To save your settings locally
- To persist tabs after browser close

**Why we request localhost permissions (optional):**
- For communication with the Companion app
- Only if you install the optional Companion
- Communication stays on your machine

### FAQ

**Q: Do you collect my browsing data?**
A: No. All data stays on your machine. We don't have servers to send it to.

**Q: Why do I need the Companion app?**
A: You don't! Local mode works without it. The Companion enables cross-browser sync.

**Q: Does this work with incognito tabs?**
A: Extension sees incognito tabs if you enable it in extension settings. Your choice.

**Q: Can I use this with one browser?**
A: Yes! Local mode organizes tabs in a single browser. Sync mode requires multiple browsers.

**Q: How much does it cost?**
A: Free forever. No ads, no premium tiers, no nonsense.
